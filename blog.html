Question-1: What are the different ways to select an element in the DOM?

1.getElementById()
2.getElementsByClassName()
3.getElementsByTagName()
4.querySelector()
5.querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Ans : innerHTML, innerText, and textContent are three different ways to access or modify the content inside a DOM element, but they behave differently. Here's how they compare:
innerHTML : Gets or sets the HTML content inside an element, including tags.
innerText : Gets or sets only the visible text inside an element (ignores hidden elements and extra spaces).
textContent : Gets or sets all text inside an element, including hidden text, without parsing HTML.

Question-3: What is event delegation in the DOM?

Ans : Event delegation is a technique in the DOM that allows you to handle events efficiently by assigning a single event listener to a parent element instead of adding multiple listeners to individual child elements. This works because of event bubbling, where an event triggered on a child element propagates up to its ancestors.

Question-4: What is event bubbling in the DOM?

Ans : Event bubbling is a type of DOM event propagation where the event first triggers on the innermost target element, and then successively triggers on the parents of the target element in the same nesting hierarchy till it reaches the outermost DOM element or document object.
